Задача 2

Есть приложение, состоящее из 2 представлений: filter_dict и sort_dict.
filter_dict работает по урлу /filter/ и фильтрует данные в списке пользователей
sort_dict работает по урлу /sort/ и сортирует список пользователей по заданным параметрам.

Если вызвать урл /filter/?gender=female, то приложение выводит список пользователей, у которых значение поля gender равно female.

Если вызвать урл /filter/?first_name=may&gender=female, то приложение выведет список пользователей, у которых значение поля gender соответствует female, а значение поля first_name - may.

Регистр переданных параметров и значений игнорируется.

Ваша задача доработать функцию sort_dict, чтобы она сортировала список пользователей, выводила фамилию и имя первого пользователя в отсортированном списке и обрабатывала следующий формат запроса:

/sort/?key=ключ&direction=направление, где

  • ключ - это название одного из полей в объекте пользователя (first_name, last_name, gender),
  • direction - это направление сортировки (asc - по возрастанию, desc - по убыванию).
Например, при вызове урла /sort/?key=first_name&direction=desc список пользователей будет отсортирован по убыванию по полю first_name, а представление вернет Larsen Solis.
